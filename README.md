Projeto de análise de dados utilizando Python (Pandas e Plotly) e Power BI, com foco no tratamento, padronização e enriquecimento de uma base real de registros de óbitos. Foram aplicadas técnicas de limpeza de dados, transformação de datas, categorização de variáveis (idade, período do dia, raça, escolaridade e causas de óbito via CID-10) e tratamento de valores nulos. Após a consolidação do dataframe final, os dados foram integrados ao Power BI para a criação de um dashboard interativo com filtros, proporcionando uma visualização clara e intuitiva dos principais indicadores e padrões demográficos e epidemiológicos.
